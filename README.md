# Mundial Front

Front Angular con PrimeNG para mostrar los 7 participantes del mundial, sus selecciones asignadas y el puntaje calculado desde JSON locales.

## Datos

- `src/assets/data/participants.json`: participantes y selecciones.
- `src/assets/data/team-results.json`: rendimiento de cada selección.
- `src/assets/data/scoring-rules.json`: reglas de puntuación.

## Ejecutar

```bash
npm install
npm run start
```

## Ajustes rápidos

- Edita `participants.json` para cambiar nombres o selecciones.
- Edita `team-results.json` para simular resultados del torneo.
- Edita `scoring-rules.json` para modificar el sistema de puntos.
